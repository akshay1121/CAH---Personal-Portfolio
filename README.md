# Creative Designer Portfolio

A static, single-page personal portfolio website created as a class assignment. It recreates the overall layout of the provided reference design using plain HTML and CSS.

## Preview

Open `index.html` in any modern web browser. No build process, server, or package installation is required.

## Features

- Full-width hero section with navigation and social icons
- Services, portfolio, profile, education, and work-experience sections
- Featured services, testimonials, client logos, contact form, and footer
- Locally stored image assets for the page content
- Font Awesome icons loaded through CDN

## Project structure

```
.
├── index.html      # Page structure and content
├── styles.css      # Page styling and layout
├── image1.png … image24.png
│                  # Images used throughout the page
└── LinkToBeCloned  # Source design reference
```

## Run locally

1. Clone or download this repository.
2. Open `index.html` in your browser.

Alternatively, serve the folder with any static-file server and visit its local address.

## Technologies

- HTML5
- CSS3
- [Font Awesome](https://fontawesome.com/) (CDN)

## Design reference

This assignment was based on the [C-Kav / Grami template demo](https://demo.templatemonster.com/demo/182745.html).

## Notes

The contact form and social-media links are presentation-only in this static version; they are not connected to a backend or external profiles.
